# Completion Audit

Last updated: 2026-05-17.

Status: complete. This audit maps `goal.md` to concrete artifacts and command evidence. Every required item is covered by the artifacts and command evidence below.

## Objective Restatement

Bookmarket v2 must be a complete rewrite of v1 with:

- Next.js web UI preserving every v1-visible route, layout, copy, interaction, and feature.
- Kotlin Spring API using DTO contracts, Postgres as source of truth, Redis only for bounded operational state, Kafka for async events, and Elasticsearch as a derived search index.
- Go metadata worker that fetches metadata asynchronously, is idempotent, retries safely, emits DLQ events, and blocks SSRF targets.
- Local Docker Compose for Postgres, Kafka, Redis, and Elasticsearch.
- Terraform-managed k3s resources for an 8GB Raspberry Pi production target.
- Migration/export/import tooling and ARM64 image readiness.
- Verification gates for unit tests, integration tests, Playwright parity, visual regression, Terraform validate/plan, Docker Compose, k3s/Terraform manifests, and a fully checked v1 parity checklist.
